# Naga Sadineni

I build AI systems that actually ship to production.

Multi-agent orchestration, graph-based RAG, LLM fine-tuning, streaming inference — I've built all of it end-to-end and deployed it at scale. I co-founded an AI startup, shipped enterprise platforms for Fortune 500 clients, and I'm most alive when I'm taking something from a whiteboard sketch to serving real users.

## What I Build

**Autonomous AI agents** — Production multi-agent systems using LangGraph state machines with tool selection, plan-execute-reflect loops, error recovery, and streaming responses. I build agents that reason over documents, browse the web, execute code in sandboxes, and coordinate with each other.

**Graph-RAG pipelines** — Not just basic vector search. I build retrieval systems that extract entities and relationships into knowledge graphs, combine semantic search with structured reasoning, and serve contextual answers from 10GB+ document corpora.

**LLM infrastructure** — Fine-tuning pipelines (LoRA/QLoRA), vLLM-optimized inference clusters on multi-GPU setups, model evaluation frameworks with RAGAS, and the CI/CD to deploy it all with zero downtime.

**Full-stack applications** — React + TypeScript frontends with streaming chat UIs, FastAPI backends, real-time data pipelines with Kafka + Redis, and the Terraform/Kubernetes infrastructure underneath.
